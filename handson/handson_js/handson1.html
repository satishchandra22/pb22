Javascript Declerarions

there are three types of declerations

  • var
  • const
  • let

var - var is a variable decleration for global variable

const - const is for constant variables. you canot change the values of const variable

let - let is for local varible which takes block scope for the perticular declared varible. it is highly recomended to use let instead of var.
