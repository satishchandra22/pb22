1. What are promises and why do we need them?

Promise objects are used to manage asynchronus operations. They are used to resolve api calls and also for call back hell. They contain three stages which are pending, resolved and rejected. We can initialize a promise with resolve and reject statements and call them using .then() and .catch() methods.

async/await key words are used for asynchronous operations. They are introdused in es8 vertion. they make promises esier to write. async used before a function. function with async keyword always returns a promise. await makes a function wait for the promise.

8. What is promise chaining

promise chaining is producing series of promises to resolve call back hell. we will be writing a function that returns a promise and will call that function where ever we need that promise. 

        let printNumber = ( time, num ) => {

                if( num ){setTimeout(()=>{resolve( num() )}, time)}
                else{reject( console.log("No number") )}
