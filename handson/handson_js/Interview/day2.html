1. Explain call(), apply() and, bind() methods. Give an example of call(), apply(), bind()

These are the different ways of invoking a function
It assumes that the first argument is object that can be passed to the function.
call: In case of call each and every argument which is required to passed to the function will be passed individually.
apply: In case of apply each and every argument which is required to be passed to the function will be passed inside an array
bind: In case of bind, a new function is created. this new function can be used anytime anywhere. 

        let obj = {
            name: 'my_name',
            age: 25
        }
        function fun(a,b){
            console.log(this.name+this.age+a+b);
        }
        const a='satish';
        const b=44;
        fun.call(obj,a,b);// apply arguments individually
        fun.apply(obj,[a,b]);// aplly arguments in array;
        const new_fun = fun.bind(obj);
        obj.new_fun(a,b);// creates a new function for 'this' reference

3. What are function constructors?

Function constructor is a kind of regular function to create multiple objects at a time. When creating function constructor, it is good practice to keep the first letter of the name, capital.
new and this keyword is mandatory to be used with function constructor. 

        function Person(name, salary){
            this.name = name,
            this.salary = salary
        }

        const person1 = new Person('name', 1000);

4. Explain prototypes

every object in javascript inherits its properties and methods from its prototype. If you want to add any property or method to a function constructor, you can add them to its prototype. 

        Person.prototype.newProp = 'new property';

5. What is prototype chain

One object can inherit properties from another object. every object inherits properties of its prototype. prototypes are also objects containing their own methods and properties. they also have another prototype object. this chain goes on and end at null object which contains no properties and methods. this is called prototype chain.

7. What are callbacks?

Call back function are the functions which are passed as an argument to an another function. 

        function my_fun(a, callBack){
            console.log(a);
            callBack();
        }

Why do we use it : The call vacks are needed because javascript is an event driven language. That means instead of waiting for a response javascript will keep executing while listening for other events.

8. What is the use of setTimeout

setTimeout method is used to invoke or execute a task after certain time or specified number of milliseconds. 

        setTimeout(()=>{
