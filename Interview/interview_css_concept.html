CSS Concepts

1. Difference between flex row and column

Flex flow is a flex container property and choosed based on how one wants to placed child elements one by one. if you place child elements in row manner, x asis becomes main axis, for column manner, y axis becomes main axis and other becomes cross axis.

2.Explain inline, internal ans external stylesheet

We can style html element in 3 ways which are inline, internal, external styling. inline styling is writting css style in style attribute of html element. it has heigst priority among other styling types.
internal styling is giving by style tag inside head tag. Internal has second heighest priority.
In external styling, external style sheet to be created and linked to html document by link tag in head of the file.

3. Justify content alowsyou todo what?

Justify content is a flexcontainer property which is used to align items according to main axis of container. there are 6 values it can take.

4. Difference between absolute and relative positioning

Relative is used to position the element relative to its original position assigned by the browser. absolute position is done relative to the nonstatic parent element.

5. What is grid-template-columns used for?

grid-template-column is a grid container property which defines number of columns and their widths. it takes width of columns and number of values given is taken as number of columns with respective width.

6. What is z-index in CSS?

using z-index property, you can define how elemnts haveto be placed in ovelaping situation. element having higher index is placed over lower index elemnet. it is applied only to nonstatic elements. ie. position property should be given either absolute or relative or fixed or sticky.

7. What is differnce between padding and margin?

these are two properties of css elements and defines cssbox model. margin defines space between border and an other element. it is the space putside the border. padding define space between content and border. margin can take both positive and negetive values. padding takes only possitive values. using margin,element size does not change, while padding changes the size of the element.

8. What is box sizing?

it defines how height and width applied to element. it takes two values, content box and border box. content box is default. it exludes the padding, border and margin whereas border box includes all these three.

tip: *{ margin:0; padding: 0; box-sizing : border-box} this has to be written in every css file.

9. What is animation delay?

it is a css animation property, it specifies how long animation should wait before it's start.

10. Which property you will use to merge celss vertically in grid?

grid column start and grid column end are used on grid container child elenent, grid-column is short hand notation for this.

11. Which property you will use to merge cells horizontally in grid?

grid row start and grid row end are used on grid container child elenent, grid-row is short hand notation for this.

12. Explain CSS box model?

it is set of rules which tells how elements should be rendered on browser. it defines four boxes content area, paddong border and margin.

13. what is the difference between display none and visibility hidden?

both are used to hide elements. visibility hidden hides without effecting its space while display none deletes the space of the element.
